# Sentry SDK for Godot

Sentry helps you track errors and crashes in your Godot games — so you can find and fix problems before your players notice.

## Quick Start

1. Sign up or log in at [sentry.io](https://sentry.io).
2. Create a new Godot project (or select an existing one).
3. Copy the DSN from **Project Settings > Client Keys (DSN)**.
4. In Godot editor, open **Project > Project Settings...** and navigate to **Sentry > Options** to paste the DSN.

5. *(Optional)* You may also turn on **Enable Logs** and **Attach Scene Tree** in the same settings section, and **Attach Screenshot** under **Sentry > Experimental** for richer diagnostics.

That's it — your game will now send errors and crashes to Sentry.

## Documentation

Full documentation is available at [docs.sentry.io/platforms/godot](https://docs.sentry.io/platforms/godot/).

You can also search for "Sentry" in **Search Help** within the Godot editor — all SDK types are prefixed with `Sentry`.

# Sentry SDK for Godot — Demo Project

This demo project showcases the features of the Sentry SDK for Godot.

## Getting Started

1. Open this project in Godot Engine.
2. Open **Project > Project Settings...** and navigate to **Sentry > Options** to paste your DSN.
3. Run the project.

Don't have a DSN yet? Sign up at [sentry.io](https://sentry.io), create a Godot project, and copy the DSN from **Project Settings > Client Keys (DSN)**.

## What's Inside

- **Capture Events** — send messages, trigger crashes, report errors, and submit user feedback.
- **Enrich Events** — add breadcrumbs, tags, custom contexts, and file attachments.

## Documentation

Full documentation is available at [docs.sentry.io/platforms/godot](https://docs.sentry.io/platforms/godot/).
